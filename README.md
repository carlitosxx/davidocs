# Produccion
Para cambiar el nombre de la aplicacion usar el paquete change_app_package_name y correr el siguiente comando:
```
flutter pub run change_app_package_name:main com.bigdavi.davidocsv2
```
Generar el bundle
```
flutter build appbundle
```
